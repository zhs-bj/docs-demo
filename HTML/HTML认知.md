## 网页认知

### 网页
1. 网页组成：文字、图片、音频、视频、超链接
2. 网页背后的本质：前端程序员写的代码，在浏览器空白页面右键点击“检查”可以看到源代码
3. 前端的代码是通过浏览器转化（渲染和解析）成页面

### 浏览器
1. 浏览器：是网页显示、运行的平台
2. 常见的五大浏览器：IE浏览器，火狐浏览器，**谷歌浏览器**，Safari浏览器，Opera浏览器
![](https://picui.ogmua.cn/s1/2026/02/25/699eabb078051.webp)
3. 渲染引擎（浏览器内核）：浏览器中专门对代码进行**解析渲染**的部分
4. 浏览器出品的公司不同，内在的渲染引擎也是不同的，渲染引擎不同，导致解析相同代码时的**速度、性能、效果**也是不同的。

### Web标准
1. 为什么需要Web标准：不同浏览器的渲染引擎不同，对于相同代码解析的效果会存在差异。Web标准让不同的浏览器按照相同的标准显示结果，让**展示的效果统一**。

2. Web标准中分成三个构成：

| 构成 | 语言 | 说明 |
| ---- | ---- | ---- |
| 结构 | HTML | **页面元素**和内容 |
| 表现 | CSS | 网页元素的外观和位置等**页面样式**（如：颜色、大小等） |
| 行为 | JavaScript | 网页模型的定义与**页面交互** |

3. Web标准要求页面实现：结构、表现、行为三层分离

## HTML的基本了解和使用

### 概念
1. HTML(Hyper Text Markup Language) 超文本标记语言
2. 专门用于网页开发的语言，主要通过**HTML**标签对网页中的文本、图片、音频、视频等内容进行描述

### HTML骨架结构
1. HTML页面固定结构：整体、头部、标题、主体
2. 网页中的固定结构通过**HTML标签**进行描述

```html
HTML基础结构示例

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>HTML基础结构示例</title>
</head>
<body>
  <h1>Hello HTML!</h1>
</body>
</html>
```

### 开发工具：VS Code
1. 创建.html文件
2. 输入：！+ 回车，html骨架就出来了
3. 快捷键：ALT+B，会将当前的.html文件在默认浏览器打开
