## 网页认知

### 网页
1. 网页组成：文字、图片、音频、视频、超链接
2. 网页背后的本质：前端程序员写的代码，在浏览器空白页面右键点击“检查”可以看到源代码
3. 前端的代码是通过浏览器转化（渲染和解析）成页面

### 浏览器
1. 浏览器：是网页显示、运行的平台
2. 常见的五大浏览器：IE浏览器，火狐浏览器，**谷歌浏览器**，Safari浏览器，Opera浏览器
<img src="https://picui.ogmua.cn/s1/2026/02/25/699eabb078051.webp" width="300" style="display:block;margin:0 auto;">
3. 渲染引擎（浏览器内核）：浏览器中专门对代码进行**解析渲染**的部分
4. 浏览器出品的公司不同，内在的渲染引擎也是不同的，渲染引擎不同，导致解析相同代码时的**速度、性能、效果**也是不同的。

### Web标准
1. 为什么需要Web标准：不同浏览器的渲染引擎不同，对于相同代码解析的效果会存在差异。Web标准让不同的浏览器按照相同的标准显示结果，让**展示的效果统一**。

2. Web标准中分成三个构成：

| 构成 | 语言 | 说明 |
| ---- | ---- | ---- |
| 结构 | HTML | **页面元素**和内容 |
| 表现 | CSS | 网页元素的外观和位置等**页面样式**（如：颜色、大小等） |
| 行为 | JavaScript | 网页模型的定义与**页面交互** |

3. Web标准要求页面实现：结构、表现、行为三层分离

## HTML的基本了解和使用

### 概念
1. HTML(Hyper Text Markup Language) 超文本标记语言
2. 专门用于网页开发的语言，主要通过**HTML**标签对网页中的文本、图片、音频、视频等内容进行描述

### HTML骨架结构
1. HTML页面固定结构：整体、头部、标题、主体
2. 网页中的固定结构通过**HTML标签**进行描述

```html
HTML基础结构示例

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>HTML基础结构示例</title>
</head>
<body>
  <h1>Hello HTML!</h1>
</body>
</html>
```

### 开发工具：VS Code
1. 创建.html文件
2. 输入：！+ 回车，html骨架就出来了
3. 快捷键：ALT+B，会将当前的.html文件在默认浏览器打开

### HTML语法规范

1. HTML的**注释**：在该行按下快捷键：ctrl + /，再按一次取消注释
2. HTML标签的结构：
- 标签由<、>、/、英文单词或字母组成。
- 标签中<>包起来的英文单词或字母是**标签名**
- 常见标签由两部分组成，**双标签**，开始标签和结束标签
- 少数标签由一部分组成，**单标签**
3. HTML标签与标签之间的关系：
- 父子关系（嵌套）
```html
<head>
    <title></title>
<head>
```
- 兄弟关系（并列）
```html
<head></head>
<body></body>
```

## HTML标签学习

### 排版标签

#### 标题标签
1. 代码:
```html
h 系列标签
<h1>1级标题</h1>
<h2>2级标题</h2>
<h3>3级标题</h3>
<h4>4级标题</h4>
<h5>5级标题</h5>
<h6>6级标题</h6>
```
2. 语义：1~6级标题，重要程度一次递减
3. 特点：
- 文字都有加粗
- 文字都有变大，1->6文字逐渐减小
- 独占一行

#### 段落标签
1. 场景：用于分段显示
2. 代码：
```html
<p>我是一段文字</p>
```
3. 特点：
- 段落之间存在间隙
- 独占一行

ps:文字太长可以在VS Code中调整为：自动换行
步骤为：File -> Preference -> Settings -> 搜索 Word Wrap -> 第一个改为on

#### 换行标签
1. 场景：让文字强制换行显示
2. 代码：
```html
<br>
```
3. 特点：
- 单标签
- 文字强制换行

#### 水平分割线标签
1. 场景：分割不同主题内容的水平线，常见于标题和主题内容之间
2. 代码：
```html
<hr>
```
3. 特点：
- 单标签
- 在页面中显示一条水平线

### 文本格式化标签

#### 文本格式化标签的介绍

1. 场景：需要让文字**加粗**，<u>下划线</u>，*倾斜*，~~删除线~~等效果
2. 代码：（双标签）

|标签|说明|               
|---|---|
|b,strong|加粗|
|u,ins|下划线|
|i,em|倾斜|
|s,del|删除线|

3. 语义：突出重要性的强调语境

#### 标签语义化

1. 实际项目开发中选择标签的原则:标签语义化
- 即:根据语义选择对应正确的标签
- 如:需要写标题，就使用h系列标签
- 如:需要写段落，就使用p标签
2. 好处:
- 对人:好理解，好记忆
- 对机器:有利于机器解析，对搜索引擎(SEO)有帮助
3. 推荐:
- strong、ins、em、del，表示的强调语义更强烈

### 媒体标签

#### 图片标签
1. 场景：在网页中显示图片
2. 代码：
```html
<img src="" alt="">
```
3. 特点：
- 单标签
- img标签需要展示对应的效果，需要借助标签的属性进行设置。
4. 属性注意点：
- 标签的属性写在开始标签内部
- 标签上可以同时存在多个属性
- 属性之间以空格隔开
- 标签名与属性之间必须以空格隔开
- 属性之间没有顺序之分

##### 图片标签的src属性
1. 属性名：src
2. 属性值：目标图片的路径
3. 若当前网页和目标图片在同一个文件夹中，路径直接写目标图片的名字（或加上./）即可（包括后缀名）

##### 图片标签的alt属性
1. 属性名：alt
2. 属性值：替换文本
- 当图片加载失败时，才显示alt的文本
- 当图片加载成功时，不会显示alt的文本
![](https://picui.ogmua.cn/s1/2026/02/25/699ec20c84323.webp)

##### 图片标签的title属性
1. 属性名：title
2. 属性值:提示文本
- 当鼠标悬停时，才显示的文本
3. 注意点:title属性不仅仅可以用于图片标签，还可以用于其他标签
![](https://picui.ogmua.cn/s1/2026/02/25/699ec2aed0ca5.webp)

##### 图片标签的width和height属性
1. 属性名:width和heig
2. 属性值:宽度和高度(数字)
3. 注意点:
- 如果只设置width或height中的一个，另一个没设置的会自动等比例缩放(此时图片不会变形)
- 如果同时设置了width和height两个，若设置不当此时图片可能会变形

#### 路径
##### 绝对路径（不常用）
1. 绝对路径：指目录下的绝对位置，可直接到达目标位置，通常从盘符开始的路径
2. 例如:
- 盘符开头: D:\day01\images\1jpg
- 完整的网络地址：https://www.itcast.cn/2018czgw/images/logo.gif

##### 相对路径(常用)
1. 概念普及（比如
- 当前文件:当前的html网页
- 目标文件:要找到的图片
2. 相对路径:从当前文件开始出发找目标文件的过程
3. 相对路径分类
- 同级目录
- 下级目录
- 上级目录

###### 相对路径-同级目录
1. 同级目录:当前文件和目标文件在同一目录中
2. 代码步骤:直接写目标文件的名字即可
```html
方法-:<img src="目标图片.gif">
方法二:<img src="./目标图片.gif">
```

###### 相对路径-下级目录
1. 下级目录:目标文件在下级目录中

2. 代码步骤:
- 先知道在哪个文件夹里面 → 文件夹名字
- 进入这个文件夹→/
- 此时看到目标文件直接喊 → 直接写目标文件名字
```html
假设图片在和当前文件夹同一级别的images文件夹中
<img src="images/目标图片.gif">
```

###### 相对路径-上级目录
1. 上级目录:目标文件在上级目录中
2. 代码步骤:
- 先出当前文件夹，到上一级目录 → ./ （上几级就几个点）
- 此时看到目标文件直接喊 →直接写目标文件
```html
<img src="./images/目标图片.gif">
```

#### 音频标签
1. 场景：在页面中插入音频
2. 代码：
```html
<audio src="./music.mp3" controls autoplay loop></audio>
```
3. 常见属性：

|属性名|功能|
|---|---|
|src|音频的路径|
|controls|显示播放的控件|
|autoplay|自动播放（部分浏览器不支持）|
|loop|循环播放|

4. 注意点：
- 音频标签目前支持三种格式：MP3、Wav、Ogg

#### 视频标签
1. 场景:在页面中插入视频
2. 代码：video标签
```html
<video src="./video.mp4" controls autoplay loop></video>
```
3. 常见属性:

|属性名|功能|
|---|---|
|src|视频的路径|
|controls|显示播放的控件|
|autoplay|自动播放(谷歌浏览器中需配合muted实现静音播放)|
|loop|循环播放|
 
 4. 注意点：
 - 视频标签目前支持三种格式：**MP4**、WebM、Ogg

### 链接标签

#### 链接标签的介绍

1. 场景:点击之后，从一个页面跳转到另一个页面
2. 称呼: a标签、超链接、锚链接
3. 代码
```html
<a href="./目标网页.html">超链接</a>
```
4. 特点:
- 双标签，内部可以包裹内容
- 如果需要a标签点击之后去指定页面，需要设置a标签的href属性
- 空链接：href设为 # 
- 自带颜色

#### 链接标签的target属性
1. 属性名:target
2. 属性值:目标网页的打开形式

|取值|效果|
|---|---|
|_self|默认值，在当前窗口中跳转(覆盖原网页)|
|_blank|在新窗口中跳转(保留原网页)|

3. 代码：
```html
<a href="https://www.baidu.com/" target="_blank">百度一下</a>
```
